Como configurar?
 Para configurar edite as linhas do arquivo formula.cgi indicadas abaixo:

     #!/usr/bin/perl
 Pergunte para seu provedor de hospedagem qual o path do PERL em seu servidor. Essa é a primeira linha do script.

      $seuemail = 'seuemail@webscript.kit.net';
Coloque o seu e-mail que vai receber os campos do formulário.
Existe possibilidade de colocar uma \ (barra) antes da @ (arroba) caso o script não funcione. Exemplo: seuemail\@webscript

      @referers = ('www.webscriptonline.kit.net','webscriptonline.kit.net');
Edite para o endereço do seu site. 

     $sendmail = '/usr/sbin/sendmail';
Modifique o path do sendmail,  programa de email do servidor.     

      $erro = "http://www.webscriptonline.net.kit/erro.html";
Edite para o endereço do seu site e crie um arquivo chamado erro.html que avisa que faltou dados a preencher no formulário ou foi preenchido indevidamente.

     $fim = "http://www.webscriptonline.kit.net/confirmado.html";
 Pagina de confirmação de envio do formulário.Edite para o endereço do seu site e crie um arquivo chamado confirmado.html     

     $Subject = "nomedosite";
Troque o termo "nome do site" pelo nome do seu site ou pelo que vai aparecer em "assunto" do e-mail.

Enviando para o servidor
Depois de tudo configurado, você deve passar os arquivos para o servidor. Vejamos então:
» Transfira o arquivo formula.cgi para o seu diretório /cgi-bin/  e dê permissão 755 (chmod 755)
» Transfira o arquivo que contém o formulário, para qualquer diretório do seu site.

Como utilizar o FormUla 2.5 ?
 Nos arquivos formulario.html e formulario2.html existem modelos de formulário. Você deverá no código fonte, mudar a linha: "http://www.webscriptonline.kit.net/cgi-bin/formula.cgi"  de acordo com seu site.

Depois cole o formulário na página de seu site e pronto. 
OBS: para os que não sabem, o formulário está entre <form> e </form> (inclusive)
